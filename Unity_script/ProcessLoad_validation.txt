
1〜3までのチェックを Static / Movable (キャラクターが多く出るなら、3の後にボーン数なども加えてSkinningでも) で行う。

=======================================================================================

1. 【Polygon 検証】

  ・1  Instance
  ・1  Material
　・100Poly / 1000Poly / 4000Poly  (※多分、ポリゴン数だけでは負荷に大きく影響しないはず)

=======================================================================================

2A. 【GameObject 検証 (20 / 40 / 80)】

  ・ 100 Poly ・ 1 Mtrl ・ 20 Obj / 100 Poly ・ 1 Mtrl ・ 40 Obj / 100 Poly ・ 1 Mtrl ・ 80 Obj 
  ・ 1000 Poly ・ 1 Mtrl ・ 20 Obj / 1000 Poly ・ 1 Mtrl ・ 40 Obj / 1000 Poly ・ 1 Mtrl ・ 80 Obj
  ・ 4000 Poly  ・ 1 Mtrl ・ 20 Obj / 4000 Poly ・ 1 Mtrl ・ 40 Obj / 4000 Poly ・ 1 Mtrl ・ 80 Obj

  → GOOD(↑45fps / 10 Draw Call)   : [ 1000 Poly ]  ・ 1 Mtrl  ・ [ 40 Obj ]  
  → OK (44〜30fps / 20 Draw Call)    : [ 4000 Poly ] ・ 1 Mtrl  ・ [ 20 Obj ]
  → NG (↓29fps  / 40 Draw Call)      : [ 4000 Poly ] ・ 1 Mtrl  ・ [ 80 Obj ]

---------------------------------------------------------------------------------------

2B. 【Instance 検証 (20 / 40 / 80 / 160 Inst)】
	
  ・1000  Polygons
  ・40 GameObject
  ・1000 Poly ・ 40 Obj ・ 20 Inst / 1000 Poly ・ 40 Obj ・ 40 Inst / 1000 Poly ・ 40 Obj ・ 80 Inst

  → GOOD (↑45fps / 40 Draw Call)   : 1000 Poly ・ 1 Mtrl  ・ 40 Obj ・ [ 20 Inst ]
  → OK (44〜30fps / 60 Draw Call)    : 1000 Poly ・ 1 Mtrl  ・ 40 Obj ・ [ 80 Inst ]
  → NG (↓29fps  / 130 Draw Call)     : 1000 Poly ・ 1 Mtrl ・ 40 Obj ・ [ 160 Inst ]

=======================================================================================

3. 【Materials 検証】

  ・1000 Polygons
  ・40   GameObjects
  ・20   Instances
  ・1000 Poly ・ 40 Obj ・ 20 Inst ・ 10 Mtrl / 1000 Poly ・ 40 Obj ・ 20 Inst ・ 20 Mtrl /
    1000 Poly ・ 40 Obj ・ 20 Inst ・  40 Mtrl / 1000 Poly ・ 40 Obj ・ 20 Inst ・ 80 Mtrl 

  → GOOD (↑45fps / 70 Draw Call)   : 1000 Poly ・ 40 Obj ・ 20 Inst ・ [ 10 Mtrl ]
  → OK (44〜30fps / 100 Draw Call)   : 1000 Poly ・ 40 Obj ・ 20 Inst ・ [ 40 Mtrl ]
  → NG (↓29fps  / 200 Draw Call)     : 1000 Poly ・ 40 Obj ・ 20 Inst ・ [ 80 Mtrl ]

=======================================================================================

4. 【Posteffect 検証】 ※ 必要なPostEffectを Materials 検証までの GOOD と OK データに載せてバリエーションを試算

  ・tonemapping
  ・colorgrading
  ・FXAA
  ・SSAO
  ・Bloom
  ・depth of field


 →GOOD (↑45fps / 100 Draw Call)  =  (1000 Poly ・ 40 Obj ・ 20 Inst ・ 10 Mtrl ) + PostEffects ( tonemapping * colorgrading * FXAA )
 →OK(44〜30fps / 110 Draw Call)   =   (1000 Poly ・ 40 Obj ・ 20 Inst ・ 10 Mtrl ) + PostEffects ( tonemapping * colorgrading * FXAA * Bloom ) or PostEffects ( tonemapping * colorgrading * SSAO * Bloom) 
 →NG(↓29fps  / 210 Draw Call)     =  (1000 Poly ・ 40 Obj ・ 20 Inst ・ 10 Mtrl ) + PostEffects ( tonemapping * colorgrading * FXAA * SSAO * Bloom)

 →GOOD (↑45fps / 90 Draw Call)  =  (1000 Poly ・ 40 Obj ・ 20 Inst ・ 40 Mtrl ) + PostEffects ( tonemapping * colorgrading )
 →OK(44〜30fps / 130 Draw Call)   =   (1000 Poly ・ 40 Obj ・ 20 Inst ・ 40 Mtrl ) + PostEffects ( tonemapping * colorgrading * FXAA) or PostEffects ( tonemapping * colorgrading * SSAO) 
 →NG(↓29fps  / 250 Draw Call)     =  (1000 Poly ・ 40 Obj ・ 20 Inst ・ 40 Mtrl ) + PostEffects ( tonemapping * colorgrading * FXAA *  SSAO )

=======================================================================================

おまけ Textures 検証(Base / Normal / Roughness・Metal ) ※絵作り、オブジェクトの種類によるので一律での試算が難しい。メモリ容量とフェッチコストの話になると思うので、決まったモデルがあるならMaterials 検証と同時に行ってしまって目安にするぐらいが良いかも?


  ・128 x 3 / 256 x 3 / 512 x 3 / 1024 x 3 

 → 128 x 3 GOOD 
 → 512 x 3 OK 
 → 1024 x 3 NG
