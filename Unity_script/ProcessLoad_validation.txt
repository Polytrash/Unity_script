2A. GameObject 検証 (20 / 40 / 80)
  ・ 100 Poly ・ 20 Obj / 100 Poly ・ 40 Obj / 100 Poly ・ 80 Obj 
  ・ 1000 Poly ・ 20 Obj / 1000 Poly ・ 40 Obj / 1000 Poly ・ 80 Obj
  ・ 4000 Poly ・ 20 Obj / 4000 Poly ・ 40 Obj / 4000 Poly ・ 80 Obj

  → GOOD : 1000 Poly ・ 40 Obj 
  → OK     : 4000 Poly ・ 20 Obj
  → NG     : 4000 Poly ・ 80 Obj
---------------------------------------------------------------------------------------

2B. Instance 検証 (20 / 40 / 80 / 160 Inst)
  ・1000  Polygons
  ・40 GameObject
  ・1000 Poly ・ 40 Obj ・ 20 Inst / 1000 Poly ・ 40 Obj ・ 40 Inst / 1000 Poly ・ 40 Obj ・ 80 Inst

  → GOOD : 1000 Poly ・ 40 Obj ・ 20 Inst
  → OK     : 1000 Poly ・ 40 Obj ・ 80 Inst
  → NG     : 1000 Poly ・ 40 Obj ・ 160 Inst

=======================================================================================

4. Materials 検証
  ・1000 Polygons
  ・40   GameObjects
  ・20   Instances
  ・1000 Poly ・ 40 Obj ・ 20 Inst ・ 10 Mtrl / 1000 Poly ・ 40 Obj ・ 20 Inst ・ 20 Mtrl /
    1000 Poly ・ 40 Obj ・ 20 Inst ・  40 Mtrl / 1000 Poly ・ 40 Obj ・ 20 Inst ・ 80 Mtrl 

  → GOOD : 1000 Poly ・ 40 Obj ・ 20 Inst ・ 10 Mtrl
  → OK     : 1000 Poly ・ 40 Obj ・ 80 Inst ・ 40 Mtrl
  → NG     : 1000 Poly ・ 40 Obj ・ 160 Inst ・ 80 Mtrl

 GOOD (↑45fps ) =  (100 Polygons * 40 Objects * 20 Instances * 10 Materials ) + PostEffects
 OK(44〜30fps )   =  (1000 Polygons * 80 Objects * 40 Instances * 40 Materials ) + PostEffects
 NG(↓29fps )      =  (4000 Polygons * 320 Objects * 800 Instances * 80 Materials) + PostEffects
